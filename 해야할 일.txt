[프로젝트 메인 언어 변경]  c++ -> javascript
C++/QT/Html/Javascript을 사용하여 프로젝트를 진행하였으나, 유튜브라는 html을 다루기 위해서 필수 불가결하게 javascript를 사용하게 되었고. webchannel을 통해 javascript와 c++통신을 하였음. 이 부분에 있어서 아주 귀찮은 작업이였고, 뼈대가 완성되고, qt ui 디자인을 준비하던 중 ui 디자인을 html/css 급으로 이쁘게 하기 위해선 QML을 사용해야하는데 이걸 또 배우자니 귀찮고, 모바일 버전까지 배포하려면 C++ 코드도 많은 부분 고쳐야 함에 있어서 많은 불편함을 느끼고, main 언어를 c++로 하고싶었지만, 어차피 사용할 수 밖에없는 javascript, css 같은 문법, 모바일 버전용을 생각했을 때 그냥 포기하고 html/css/javascript로 코딩하는게 모든 방면에서 편하다고 생각하여, 언어를 변경.
새로운 언어를 제대로 사용하기 위해서 html문법, css문법, javascript 동작원리 등을 공부하면서 프로젝트를 진행하고 있기에 조금 늦어지고 있음.
일단, A폴더는 현재 진행하고있는 html용 폴더, Almighty 폴더는 c++로 제작하던 폴더.
index.html을 켜보면 대충 ui확인 할 수 있음. 내부적인 javascript코드는 내부적으로 만들고있음.
이제 ui 디자인은 html/css문법만 알면 쉽게 변경 할 수 있고.
프로토타입 완성품으론 예상으로 일주일 정도 더 걸릴듯.
